# Crítica Combining Predictions for Accurate Recommender Systems

En este paper se presenta el 'ensemble method' para la predicción en sistemas recomendadores. Este metodo se basa en el resultado de diversos algoritmos para obtener una predicción final. Para esto utiliza la base de datos de netflix junto a algoritmos como Knn user-user, Knn item-item, SVD, entre otros y luego se utiliza una combinación lineal de los resultados de manera que se minimize el RMSE utilizando distintos acercamientos como redes neuronales o regresiones.

El método propuesto en este paper es fácil de entender y tiene una aplicación bastante directa y sencilla. A pesar de esto el cambio en el rendimiento es notorio y puede permitir una disminución significativa en el error, lo que es muy díficil utilizando un único método. Además, lo más destacable de este procedimiento es que es escalable para el futuro. Es decir, si más adelante se descubre un algoritmo que supera el rendimiento de los algoritmos contemporaneos, siempre será posible aumentar el rendimiento de este ( en el peor de los casos se mantendrá) utilizando un ensemble method.

Me parece que este paper logra mostrar de manera muy amplia los resultados obtenidos y realiza comparaciones claras entre los distintos enfoques que se utilizarón para realizar el 'bending'. Me hubiera gustado visualizar una forma más gráfica de la complejidad que tiene utilizar este método en termino de tiempos en comparación a los filtros colaborativos por si solo, ya que finalmente el mayor problema de esta propuesta se basa en el costo de tiempo y memoria. Además, podría ser interesante probar esto para distintas bases de datos o utilizando distintos algoritmos de recomendación, de manera que se pueda determinar si ciertos algoritmos se pueden excluir a la hora de hacer un bending dado que su aporte no es significativo y así reducir los costos.
